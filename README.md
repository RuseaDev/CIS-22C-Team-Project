## Project layout

```
include/             # Header files
src/                 # C++ source files
docs/                # Diagrams and project documentation
```

## Execute commands

```
make                  # Compiles the project
make run              # Compiles and runs the project
make clean            # Removes the compiled executable
./program             # Runs the project (no recompiling)
```

When the program asks for the input file, use:

```
sample_input.txt
```
